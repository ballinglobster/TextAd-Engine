TextAd Engine

Welcome to the TextAd Engine.

This is a simple engine for creating text based adventure games, no programming required.

Features include:

  • Image support: Display images to enhance your story.
  • Commands are already built-in for easy interaction.
  • Easy to customise and extend for programmers.
  • Open source and free to use.
Some more information on point 3:
  • Commands are modifiable inside the JavaScript file and you can always create your own commands.
  • New features can be added by editing the JavaScript file. There is no limit to what you can create.
  • The HTML and CSS files can also be customised to change the appearance and layout of your game.
